# Калибри — структура проекта

Файл `kalibri 1.jsx` был разбит на отдельные файлы, чтобы было удобнее редактировать.

## Структура

- `App.jsx` — корневой компонент (вся логика состояния приложения: чаты, звонки, панели).
- `constants.js` — общие константы: эмодзи, обои чата, цвета аватаров, папки, иконки разделов курса.
- `styles.css` — все стили приложения (обычный CSS, инжектится в App через `<style>{STYLES}</style>`).
- `assets/logo.js` — логотип (base64-картинка), вынесен отдельно, чтобы не раздувать другие файлы.
- `utils/helpers.js` — вспомогательные функции: цвет аватара, инициалы, время, id, превью чата, подсветка поиска.
- `data/seedState.js` — стартовые демо-данные (чаты, звонки, истории, темы).
- `components/*.jsx` — каждый визуальный компонент в своём файле (Avatar, ChatRow, Message, SidePanel, ChatView, CourseChat, PointTopics, CallScreen и т.д.).
- `components/index.js` — общий файл, через который компоненты импортируют друг друга.

## Как редактировать

Открывайте нужный файл компонента и правьте его — не нужно искать нужный кусок в 2700-строчном файле.

Например:
- логика и вёрстка сообщения → `components/Message.jsx`
- боковая панель настроек → `components/SidePanel.jsx`
- список чатов слева → `components/ChatRow.jsx` (одна строка) и `App.jsx` (сам список и фильтры)
- стартовые тестовые данные (кто с кем переписывается) → `data/seedState.js`
- цвета, обои, эмодзи, папки → `constants.js`

## Как собрать превью после правок

Проект собирается через esbuild (упакован локально в `node_modules/`, никаких внешних загрузок не требуется):

```
node_modules/.bin/esbuild src/entry.jsx --bundle --loader:.css=text --loader:.jsx=jsx --loader:.js=jsx --jsx=automatic --format=iife --outfile=dist/bundle.js
```

Либо просто пришлите отредактированные файлы обратно — соберу и пришлю обновлённое превью.

Иконки: пакета `lucide-react` в оффлайн-песочнице нет, поэтому в `node_modules/lucide-react` лежит локальная замена (шим) на основе уже установленного `react-icons` (набор иконок Lucide, `react-icons/lu`) — API (`size` и т.п.) идентичен, все импорты `from "lucide-react"` в коде работают как есть.
